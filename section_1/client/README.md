### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/albiancdp/Travelio_Test.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Run project for development port 4000
   ```sh
   npm run start:dev
   ```
